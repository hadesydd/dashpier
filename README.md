- **Models**: Advanced language models for email generation

- **Pattern**: Clean Architecture

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (or npm/yarn)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/mynamail.git
cd mynamail

# Install dependencies
pnpm install

# Run development server
pnpm dev
```

### Environment Variables

Create a `.env.local` file:

```env
# For email generation (example - configure based on your AI provider)
OPENAI_API_KEY=your_api_key_here
```
